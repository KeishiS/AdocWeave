= AdocWeave

AdocWeaveは、AsciiDoc文書の確認、整形、HTML変換を行うRust製ツールである。
コマンドライン、WebAssembly、Language Serverを提供する。

== 機能

* AsciiDocを安全なHTMLへ変換する。
* 文法やリンク、表などの問題を行番号付きで表示する。
* 意味を変えずに、対応済みの範囲を整形する。
* Zedなどのエディターで診断、補完、定義への移動を提供する。
* ブラウザーや別のアプリケーションに組み込む。

対応状況はxref:docs/syntax-support.adoc[対応している書式]を参照。AdocWeaveはAsciiDocの全機能を実装しない。
対応範囲と他の実装との差異はxref:docs/compatibility.adoc[AsciiDocとの違い]に記載する。

== 使用方法

GitHub Releasesからの導入方法はxref:docs/release-installation.adoc[インストールと更新]を参照。
Nix環境ではxref:docs/nix-installation.adoc[Nixで使う]を用いる。

基本的な実行例:

[source,console]
----
# HTMLへ変換する
adocweave convert manual.adoc > manual.html

# 問題を確認する
adocweave check manual.adoc

# 整形が必要かだけを確認する
adocweave format --check manual.adoc
----

全オプション、標準入力、`include::`の扱いはxref:docs/command-line.adoc[コマンドラインの使い方]を参照。

== 開発環境

Nix環境では、リポジトリのルートで次を実行する。

[source,console]
----
nix develop
cargo make verify
----

`cargo make verify`は、整形、型検査、テスト、ブラウザー向けテスト、文書検査を実行する。
個別のコマンドはxref:docs/testing.adoc[テストの実行]を参照。
