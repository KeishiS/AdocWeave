= AdocWeave

AdocWeaveは、AsciiDoc文書を確認、整形、HTML変換できるRust製のツールです。
コマンドライン、WebAssembly、Language Serverを提供します。まずはCLIで文書を変換したり、
書式やリンクの問題を確認したりできます。

== できること

* AsciiDocを安全なHTMLへ変換する。
* 文法やリンク、表などの問題を行番号付きで表示する。
* 意味を変えずに、対応済みの範囲を整形する。
* Zedなどのエディターで診断、補完、定義への移動を提供する。
* ブラウザーや別のアプリケーションに組み込む。

対応している書式と、まだ対応していない書式はxref:docs/syntax-support.adoc[対応している書式]で確認できます。
AdocWeaveはAsciiDocの全機能を実装するものではありません。対応範囲と他の実装との違いは
xref:docs/compatibility.adoc[AsciiDocとの違い]にまとめています。

== はじめに使う

GitHub Releasesから自分の環境に合うファイルを入手する方法は、
xref:docs/release-installation.adoc[インストールと更新]を参照してください。Nixを使う場合は
xref:docs/nix-installation.adoc[Nixで使う]が利用できます。

インストール後は、次のように使えます。

[source,console]
----
# HTMLへ変換する
adocweave convert manual.adoc > manual.html

# 問題を確認する
adocweave check manual.adoc

# 整形が必要かだけを確認する
adocweave format --check manual.adoc
----

コマンドの全オプション、標準入力、`include::`の扱いはxref:docs/command-line.adoc[コマンドラインの使い方]を参照してください。

== 開発

Nixを使う場合は、リポジトリのルートで次を実行すると必要なツールを用意できます。

[source,console]
----
nix develop
cargo make verify
----

`cargo make verify`は、整形、型検査、テスト、ブラウザー向けテスト、文書の検査をまとめて実行します。
個別の作業に使うコマンドはxref:docs/testing.adoc[テストの実行]に載せています。
