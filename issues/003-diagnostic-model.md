# 共通診断モデル

## 目的

構文解析、Lint、CLI、Language Serverで共有できる診断と修正の表現を定義する。

## 実装範囲

- 診断コード、重要度、メッセージ、範囲を表す型を追加する。
- 安全なText Editを保持するFix型を追加する。
- 診断の安定した並び順を定義する。
- CLI向けの人間可読形式とJSON形式を用意する。

## 完了条件

- 同じ診断集合から常に同じ順序の出力を得られる。
- 診断範囲を行番号と列番号で表示できる。
- 重複または交差するEditを検出できる。

## 検証

```console
cargo test diagnostic
```
