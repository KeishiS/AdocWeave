= インライン等幅書式の境界規則
:status: planned
:depends-on: 154

== 目的

単一バッククオートによる等幅書式の境界判定を、Asciidoctorの規則に合わせる。現在は英数字だけを
語の一部として扱うため、アンダースコアや`:`、`;`、`}`に隣接した書式を誤って認識する。

== 作業

* 単一バッククオートの開始・終了判定で、Unicodeの文字・数字・`_`を語の一部として扱う。
* 開始記号の直前が`:`、`;`、`}`の場合は、二重バッククオートだけを認識する。
* 二重バッククオートによる非制約形式は、単語中・CJK文字・絵文字に隣接しても認識する。
* 対応書式の文書に、単一と二重の使い分けを記載する。

== 対象外

`+...+`を使う文字どおりの等幅表示と、等幅部分での置換規則は別の作業として扱う。

== 完了条件

* `snake_`、`key:`、`x}`に単一バッククオートを続けた入力を等幅書式にしない。
* 同じ位置の二重バッククオート、CJK文字、絵文字に隣接する二重バッククオートを等幅書式にする。
* Asciidoctorの公式例に対応する単体テストと共通テスト用入力を追加する。
