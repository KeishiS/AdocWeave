= 最小Formatter
:status: completed
:depends-on:

== 目的

プレーンテキスト領域を安全かつ冪等に整形する。

== 実装範囲

- 改行コードを設定された形式へ統一する。
- 行末空白を削除する。
- 文書末尾の改行を統一する。
- ブロック間の過剰な空行を削減する。
- CSTを利用し、変更対象外と未対応領域のバイト列を保存する。
- 整形結果または重複しないText Editを返すAPIを定義する。

== 完了条件

- `format(format(source)) = format(source)` が成立する。
- `parse(format(source))` と `parse(source)` のsemantic ASTが意味同値である。
- 変更対象外の範囲がバイト単位で一致する。
- `format` が標準出力へ結果を出せる。
- `--check` 相当の非破壊な確認方法がある。

== 検証

[source,console]
----
cargo test formatter
cargo run -- format fixtures/format/basic.adoc
----
