# 等幅文字

## 目的

等幅文字の解析、HTML変換、未閉鎖診断を縦方向に実装する。

## 実装範囲

- 対応する区切り記号と境界条件を仕様化する。
- `Inline::Monospace` を追加する。
- HTMLの `code` 要素へ変換する。
- 閉じられていない構文を位置付きで診断する。
- 未閉鎖時は安全なテキストとして回復する。

## 完了条件

- 正常、空内容、複数個、未閉鎖を扱える。
- ブロック境界を越えて終端を検索しない。
- 等幅内容のHTML特殊文字をエスケープする。
- Formatterは等幅文字内部を変更しない。

## 検証

```console
cargo test monospace
```
