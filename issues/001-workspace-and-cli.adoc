= Cargo workspaceとCLIの初期化

== 目的

AsciiLoomを小さく実行・検証できるRust workspaceとして初期化する。

== 実装範囲

- ルートのCargo workspaceを作成する。
- `asciiloom` ライブラリ兼CLI crateを作成する。
- `convert`、`check`、`format` サブコマンドの骨格を用意する。
- fixtureの配置規約と最小サンプルを追加する。
- 入力は標準入力またはファイルから読み、出力は標準出力へ書く。

== 完了条件

- 各サブコマンドが `--help` を表示できる。
- `convert` が入力を読み込んで終了できる。
- ライブラリとCLIの責務が分離されている。
- 不正UTF-8とI/Oエラーが利用者向けエラーになる。

== 検証

[source,console]
----
cargo fmt --check
cargo clippy --all-targets --all-features
cargo test --workspace
cargo run -- convert fixtures/plain/basic.adoc
----
