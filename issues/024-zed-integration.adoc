= Zed Editor連携

== 目的

AsciiLoom Language ServerをZed Editorから起動し、初期編集支援を利用可能にする。

== 実装範囲

- `.adoc` と `.asciidoc` を認識する最小拡張または設定を追加する。
- Language Serverの起動コマンドとシェル環境の受け渡しを実装する。
- サーバー本体をZed固有コードへ依存させない。
- 動作確認用のサンプル文書と手順を追加する。

== 完了条件

- ZedからLanguage Serverを起動できる。
- 編集に応じて診断が更新される。
- 文書シンボル、Code Action、Format Documentが動作する。
- 日本語と絵文字を含む診断位置が正しい。
- Language Server異常終了がエディター本体へ波及しない。

== 検証

自動LSP統合テストに加え、Zed上でチェックリストに従って手動確認する。

2026-07-20にLinux版Zedで、言語認識、Language Server起動、診断、文書シンボル、
Code Action、Format Document、Hoverおよび補完を手動確認した。ZedでLSP文書シンボルを
Outlineへ表示するには、AsciiDoc言語設定の`document_symbols`を`on`にする必要がある。
具体的な設定と操作はxref:../editors/zed/README.adoc[Zed開発用拡張]に記載する。
