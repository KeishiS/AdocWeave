= ソースコードブロック
:status: completed
:depends-on:

== 目的

言語指定付きソースコードブロックを解析し、HTML、Lint、補完用情報を提供する。

== 実装範囲

- ソースブロックの属性行と区切りを仕様化する。
- 言語名とコード本文の範囲をASTへ保持する。
- `pre` と `code` を使った安全なHTMLを生成する。
- `missing-source-language` を実装する。
- 言語名候補を返すエディター非依存APIを追加する。

== 完了条件

- 言語指定あり、なし、空ブロック、未閉鎖を扱える。
- 言語名を安全なclass属性へ変換する。
- コード本文をインライン解析・整形しない。
- 言語指定漏れを正確な範囲で報告する。

== 検証

[source,console]
----
cargo test source_block
----
