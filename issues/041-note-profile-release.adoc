= ホスト非依存コアプロファイルのリリース検証

== 目的

AdocWeaveの標準構文プロファイルをネイティブ、WASM、CLIおよびLanguage Serverで
一貫して公開できる状態にする。特定ノートアプリの製品リリースは対象外とする。

== 実装範囲

* 必須構文、未対応構文、厳格・寛容モードおよび既定上限をversion付きで固定する。
* 公開API、診断、HTML契約、WASM APIおよびcrateのSemVer方針を文書化する。
* 処理系更新時にHTML、検索projectionおよび参照projectionを再構築すべき条件を定義する。
* 文書内・文書間xref、通常リンク、任意スキームおよびブラウザプレビューを受け入れ試験する。
* Zed、ブラウザおよび将来のネイティブホストで再利用する手動確認項目を整備する。

== 完了条件

* プロファイル必須構文が縦方向に実装・検証されている。
* ネイティブとWASMが同じ入力から同じ`Analysis`相当結果とHTMLを得る。
* 互換性を破る変更と再構築が必要な変更を利用者が判定できる。
* コアに特定DB、ノートメタデータ型またはアプリ固有サーバ実装が含まれない。
* 全workspace、WASM、LSPおよび適合性テストが成功する。

== 検証

[source,console]
----
cargo make verify
----
