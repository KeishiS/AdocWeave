= LSP文書シンボル

== 目的

共通の文書シンボル木をLSPのDocument Symbolとして公開する。

== 実装範囲

- `textDocument/documentSymbol` を実装する。
- 文書タイトルと見出しを階層付きシンボルへ変換する。
- 選択範囲と要素全体の範囲を区別する。
- 不完全な見出しを安全に扱う。

== 完了条件

- CLIとLSPのシンボル階層が一致する。
- Unicodeを含む見出しの範囲が正しい。
- 空文書では空のシンボル集合を返す。

== 検証

[source,console]
----
cargo test -p asciiloom-lsp document_symbols
----
