= AdocWeaveへの全面改名
:status: completed
:depends-on:

== 目的

公開APIを確定する前に、製品、crate、実行ファイルおよびエディター統合の名称を
`AdocWeave`へ統一する。

== 実装範囲

- 製品表示名を`AdocWeave`へ変更する。
- コアcrateとCLIを`adocweave`へ変更する。
- Language Server crateと実行ファイルを`adocweave-lsp`へ変更する。
- Zed拡張のID、crate、表示名、Language Server IDおよび実行コマンドを変更する。
- crateディレクトリ、workspace設定、cargo-makeタスク、lockfileを新名称へ揃える。
- ソース、fixture、診断source、ドキュメントおよびIssue計画から旧名称を除去する。
- リモートリポジトリを改名した後に使用するメタデータURLを
  `https://github.com/KeishiS/AdocWeave`へ変更する。

== 完了条件

- `adocweave --version`が新名称を表示する。
- Zed拡張が`adocweave-lsp`を要求し、拡張crateとLanguage Serverを型検査できる。
- `cargo metadata`に旧crateまたは旧実行ファイルが現れない。
- Git管理対象のパスと内容に旧名称が残らない。
- 全workspaceとZed拡張の検証が成功する。

== 検証

[source,console]
----
cargo make verify
git grep -i -e 'ascii''loom' -e 'ascii''[[:space:]]+loom'
----
