# インライン構文の基盤

## 目的

段落と見出しの本文を、元範囲を持つインライン要素として表現する。

## 実装範囲

- `Inline::Text` を含むインラインASTを定義する。
- 段落と見出しの本文をインライン解析器へ渡す。
- ブロック境界を越えない走査器を実装する。
- 深さ上限を設定できる解析コンテキストを用意する。

## 完了条件

- インライン構文を含まない文書のHTML出力が変更前と一致する。
- テキスト要素が元入力の範囲を保持する。
- 空文字列とUnicode入力でpanicしない。

## 検証

```console
cargo test inline_text
cargo test html_regression
```
