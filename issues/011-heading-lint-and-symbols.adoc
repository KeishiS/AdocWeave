= 見出しLintと文書シンボル
:status: completed
:depends-on:

== 目的

見出し階層を検査し、エディター非依存の文書シンボルを生成する。

== 実装範囲

- `invalid-heading-level` を実装する。
- `duplicate-heading-id` を実装する。
- 見出し記号後の空白欠落に診断と安全なFixを付ける。
- 見出し階層から文書シンボル木を生成する。
- シンボルをCLIからJSON表示できるようにする。

== 完了条件

- レベル飛び越しと重複IDを正確な範囲で報告する。
- Fix適用後に見出しの意味が変わらない。
- 文書シンボルの親子関係が見出し階層と一致する。

== 検証

----
cargo test heading_lint
cargo test document_symbols
