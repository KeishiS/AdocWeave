# 強調

## 目的

強調構文の解析、HTML変換、未閉鎖診断を実装する。

## 実装範囲

- 強調記号の開始・終了条件を仕様化する。
- `Inline::Strong` を追加する。
- HTMLの `strong` 要素へ変換する。
- 通常文字として扱う曖昧な入力をfixtureで固定する。
- 未閉鎖時に後続要素を失わない回復処理を追加する。

## 完了条件

- 強調と通常の記号を区別できる。
- 複数の強調を一つの段落で扱える。
- 未閉鎖診断が正確な開始位置を指す。
- 既存の等幅文字を壊さない。

## 検証

```console
cargo test strong
cargo test inline_regression
```
