= タイトルと見出しの解析

== 目的

文書タイトルとレベル1から5の見出しを位置情報付きASTへ追加する。

== 実装範囲

- `= Title` を文書タイトルとして認識する。
- `==` から `======` をレベル1から5の見出しとして認識する。
- 記号、区切り空白、本文の範囲を保持する。
- 空タイトル、空白欠落、過剰な記号を診断可能な形で保持する。
- 次の見出しをエラー回復の同期点として扱う。

== 完了条件

- 各レベルと文書タイトルを正しく区別する。
- 行頭以外の `=` を見出しとして扱わない。
- 不完全な見出しの後にある正常なブロックを解析できる。
- ASTスナップショットに範囲が含まれる。
