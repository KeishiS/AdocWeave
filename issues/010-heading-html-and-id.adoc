= 見出しHTMLとID生成

== 目的

タイトルと見出しをHTMLへ変換し、決定的な見出しIDを生成する。

== 実装範囲

- 文書タイトルの出力有無を設定可能にする。
- 見出しを `h1` から `h5` 相当へ変換する。
- 見出しID生成規則を文書化して実装する。
- ASCII、日本語、空白、記号を含む見出しを扱う。

== 完了条件

- 同じ見出しから常に同じIDを生成する。
- タイトルと見出し本文をHTMLエスケープする。
- 完全HTMLとHTML断片の設定を区別できる。
- 意図的なAsciidoctorとの差異をfixtureに記録する。

== 検証

[source,console]
----
cargo test heading_html
cargo test heading_id
----
