= 汎用文書projection

== 目的

HTMLとは独立して、ホストが検索や参照索引へ利用できる情報を同じ`Analysis`から
決定的に抽出する。ノートアプリ固有のDBスキーマやメタデータ型は実装しない。

== 実装範囲

* タイトル、アンカー、通常リンク、xrefおよび検索用可読テキストの汎用projectionを定義する。
* 参照元`SourceId`、範囲、`ReferenceKey`および任意の解決結果からなるedge表現を定義する。
* コード、数式、属性行および不可視構文の検索テキスト上の扱いを仕様化する。
* 抽出結果に元範囲とprojection契約versionを保持する。
* HTMLとprojectionが同じ`Analysis`を読み、独自に再解析しない。
* 順序、重複処理およびシリアライズ表現を固定する。

== 完了条件

* 一つの`Analysis`からHTMLと全汎用projectionを生成できる。
* 同じ入力と設定からバイト単位で安定した結果を得られる。
* 文書内、文書間および任意スキームxrefを同じedge型で表現できる。
* 外部URLはxrefと型で区別される。
* ノートID、タグ型、DB列またはアプリ固有必須属性をコアへ導入しない。
