= サーバ実行・キャッシュ・観測性

== 目的

コア処理系をWebサーバから安全に実行するためのホスト統合層を実装する。

== 実装範囲

- CPU解析を非同期I/Oスレッドから分離する、有界ワーカープール用アダプターを実装する。
- 並列数、待ち行列、処理時間、入力・出力・診断数等の上限を設定可能にする。
- クライアント切断、タイムアウトおよび新しい文書版で解析をキャンセルする。
- SHA-256生バイト列、構文プロファイル、全設定および処理系バージョンからキャッシュキーを生成する。
- 成功、失敗、上限超過、キャンセル、処理時間、入力サイズ、診断数およびキャッシュ結果を本文なしで観測可能にする。
- panicをリクエスト境界で隔離し、安定したエラーコードへ変換する。

== 完了条件

- 過負荷時に無制限のタスクまたはメモリを生成しない。
- キャンセル済み結果または古い文書版を保存しない。
- 設定または処理系バージョンが異なる結果をキャッシュ共有しない。
- 通常ログ、メトリクスおよびトレースに入力本文を記録しない。

== 検証
