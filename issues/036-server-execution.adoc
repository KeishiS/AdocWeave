= ホスト実行境界とキャッシュ契約

== 目的

外部サーバ、デスクトップアプリその他のホストがAdocWeaveを安全に実行するための
ホスト非依存契約と補助型を定義する。WebサーバやDBアダプター自体は実装しない。

== 実装範囲

* CPU解析を有界ワーカーへ渡すための入力、キャンセルおよび結果所有権を仕様化する。
* 生入力hash、構文プロファイル、全設定、コアAPI versionからなるキャッシュキーを定義する。
* 古い文書versionとキャンセル済み`Analysis`を公開しない規則を定義する。
* 処理時間、入力・出力サイズ、診断数および結果分類を、本文なしで観測する項目を定義する。
* panic隔離はホスト境界の責務とし、コアの通常入力エラー処理と混同しない。

== 完了条件

* 契約が特定async runtime、HTTP framework、DBまたはノートアプリへ依存しない。
* 設定または処理系versionが異なる結果をキャッシュ共有しない。
* キャンセル済みまたは古い文書versionの結果を採用しない手順をfixtureで確認できる。
* 通常ログ、メトリクスおよびトレースに入力本文を要求しない。

== 検証

[source,console]
----
cargo test --workspace cache_key
cargo test --workspace cancellation
----
