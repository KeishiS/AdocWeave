# リテラルブロック

## 目的

区切り付きリテラルブロックを解析し、安全なHTMLへ変換する。

## 実装範囲

- リテラルブロックの開始行と終端行を仕様化する。
- `Block::Literal` を追加する。
- 内容をインライン解析せず `pre` 要素へ変換する。
- ファイル末尾と次の見出しを用いたエラー回復を実装する。
- 未閉鎖ブロックを `unclosed-block` として診断する。

## 完了条件

- 空ブロック、複数行、区切りに似た本文を扱える。
- 内容中のHTML特殊文字をエスケープする。
- 未閉鎖ブロックでも取得可能なASTを返す。
- Formatterはブロック本文を変更しない。

## 検証

```console
cargo test literal_block
```
