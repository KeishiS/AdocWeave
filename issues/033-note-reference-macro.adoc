= note参照マクロとResolver境界

== 目的

`note:<uuid>#<anchor>[label]` を解析し、DB非依存の参照情報とホスト注入型Resolverを提供する。

== 実装範囲

- UUID、任意アンカー、表示文字列、エスケープおよび不完全入力の文法を定義する。
- ノートID、アンカーおよび表示文字列を独立した範囲としてASTへ保持する。
- 構文検証と、ノート・アンカーの存在確認を分離する。
- 候補検索、UUID解決およびアンカー解決の非同期Resolverインターフェースをホスト層に定義する。
- コア解析はResolverを呼ばず、補完、ホバーおよび定義移動の問い合わせ情報だけを返す。
- 不正UUID、リンク切れおよびアンカー不在の診断コードを定義する。

== 完了条件

- Resolverなしでも構文解析、HTML生成および参照抽出が決定的に動作する。
- Resolver結果を信頼せず、HTML出力前にURLポリシーを適用する。
- 入力中のマクロから過度に点滅する診断を出さない寛容モードfixtureがある。
- CLI、LSP、サーバおよびWASMが同じ参照型を利用できる。
