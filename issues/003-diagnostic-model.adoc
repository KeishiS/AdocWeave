= 共通診断モデル
:status: completed
:depends-on:

== 目的

構文解析、Lint、CLI、Language Serverで共有できる診断と修正の表現を定義する。

== 実装範囲

- 診断の安定したID、コード、重要度、メッセージ、主範囲および関連範囲を表す型を追加する。
- 安全なText Editを保持するFix型を追加する。
- 診断の安定した並び順を定義する。
- 解析失敗、上限超過、キャンセルおよび内部不整合を区別する安定したエラーコードを定義する。
- CLI向けの人間可読形式とJSON形式を用意する。

== 完了条件

- 同じ診断集合から常に同じ順序の出力を得られる。
- 診断範囲を行番号と列番号で表示できる。
- CLI、LSP、Web APIおよびWASMで同じコードと範囲を利用できる。
- 重複または交差するEditを検出できる。
