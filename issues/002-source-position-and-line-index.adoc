= ソース位置と行索引
:status: completed
:depends-on:

== 目的

解析、診断、LSPで共有するバイト範囲と行索引を実装する。

== 実装範囲

- `TextSize` と半開区間の `TextRange` を定義する。
- すべてのソース範囲を元のUTF-8バイト列上の半開区間 `[start, end)` と定義し、行・列を0起点とする。
- UTF-8バイト位置から行番号と列番号を取得する `LineIndex` を実装する。
- UTF-8コード単位およびUTF-16コード単位によるLSP位置との相互変換を実装する。
- LF、CRLF、UTF-8 BOM、NULおよびタブの位置上の扱いを仕様化する。
- 不正な文字境界と範囲外位置をエラーとして扱う。

== 完了条件

- ASCII、日本語、絵文字、結合文字で位置変換できる。
- LFとCRLFのいずれでも正しい行位置を返す。
- BOM、NULおよびタブを含む入力でも元バイト範囲を変えない。
- 文書先頭、行末、文書末尾を扱える。
- 往復可能な位置では変換の往復が恒等になる。
