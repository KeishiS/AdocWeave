= ホバーと補完

== 目的

初期構文に対する小規模で確実な編集支援を提供する。

== 実装範囲

- 見出しにレベルと生成IDを示すホバーを追加する。
- ソースコードブロックの言語名補完を追加する。
- カーソル位置からAST要素を検索する共通APIを追加する。
- 未対応位置では空の結果を返す。

== 完了条件

- 見出し記号、本文、範囲外の位置を区別できる。
- 言語名の入力途中で候補を返せる。
- コード本文や通常段落で不要な候補を返さない。
- Unicode位置から正しいAST要素を検索できる。

== 検証

[source,console]
----
cargo test -p asciiloom-lsp hover
cargo test -p asciiloom-lsp completion
----
