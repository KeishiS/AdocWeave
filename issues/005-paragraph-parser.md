# 段落の構文解析

## 目的

通常テキストと空行から位置情報付きの段落ASTを生成する。

## 実装範囲

- 文書、段落、テキストの最小ASTを定義する。
- 連続する非空行を一つの段落として解析する。
- 一つ以上の空行を段落の区切りとして扱う。
- ASTの各要素に元入力の範囲を保持する。

## 完了条件

- 空入力、1行、複数行、複数段落を解析できる。
- 先頭と末尾の空行を安全に扱える。
- 不完全な入力でもpanicしない。
- fixtureからASTスナップショットを生成できる。

## 検証

```console
cargo test paragraph
cargo run -- convert fixtures/plain/basic.adoc
```
