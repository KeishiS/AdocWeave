= 明示アンカーと安定ID

== 目的

ノート内参照に利用できる明示アンカーと、互換性を管理できる自動見出しIDを実装する。

== 実装範囲

- 明示アンカーの構文、文字集合、エスケープおよび対象ブロックとの関連付けを規範化する。
- Unicode正規化を行う箇所と、元表記を保持する箇所を定義する。
- 自動見出しIDの生成、空ID、衝突および重複時の挙動を定義する。
- 明示IDを自動IDより優先し、安定したアンカー索引をASTから生成する。
- 重複または不正アンカーを位置付きで診断する。

== 完了条件

- 同じ入力とプロファイルから同じアンカー索引を生成する。
- 日本語、結合文字、絵文字および大小文字を含む境界fixtureがある。
- HTMLのIDと参照解決用IDが仕様どおり一致する。
- 見出しID規則の変更を出力契約の変更として検出できる。
