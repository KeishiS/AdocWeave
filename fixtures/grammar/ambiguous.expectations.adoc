= 期待する解釈

- `word`内部のバッククォートと`some_identifier`は通常テキストとして残る。
- 次の行は斜体と等幅の子要素を含む一つのstrong spanになる。
- 閉じていない斜体は`unclosed-inline`を報告し、後続の等幅spanは解析される。
- リテラルブロックのインラインマーカーと`<tag>`は不透明である。`.....`は終了区切り
  ではなく内容である。
- ソースブロックのインラインマーカーと`<tag>`は不透明であり、言語は`rust`である。
- 最後のリテラルブロックは`unclosed-block`を報告し、見出しノードとして残る
  `== Recovered heading`の前で回復する。
- CSTはfixtureをバイト単位で復元する。

完全なCST、semantic ASTおよび診断の期待値は、それぞれ`ambiguous.cst`、
`ambiguous.ast`および`ambiguous.diagnostics.json`に固定する。インライン子要素の構造は
統合テストで追加検証する。
