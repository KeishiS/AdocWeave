= Fixture

fixtureは担当機能ごとに分類する。各機能ディレクトリには次のファイルを配置できる。

- `*.adoc`: ソース文書
- `*.html`: 期待するHTML断片
- `*.diagnostics.json`: 期待する診断
- `*.formatted.adoc`: 期待するFormatter出力

各fixtureは一つの動作へ集中させる。内容を表す名前を使い、正常、不正、不完全、境界値の
入力を別ファイルとして追加する。

`release/core.adoc`は機能横断のコアリリースfixtureである。期待HTMLを同じディレクトリへ
置き、CLI、Lint、Formatterのテストで同じソースを使う。
