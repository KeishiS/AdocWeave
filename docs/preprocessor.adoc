= プリプロセッサとresource snapshot

== 責務境界

AdocWeaveの通常解析はファイルシステムやネットワークへアクセスしない。hostは必要なresourceを
取得し、targetをkeyとする`ResourceSnapshot`を構築してから`preprocess`または
`preprocess_and_analyze`へ渡す。この分離により、nativeとWASMは同じsnapshotに対して同じ結果を
生成する。

hostは`discover_includes`で完全かつescapeされていないinclude directiveをI/Oなしに列挙できる。
これは条件分岐内を含むresource候補のsupersetであり、条件評価と安全性判定は`preprocess`が行う。

`base_uri`はroot文書の相対targetへ決定的に前置される論理的なsnapshot keyであり、コアが参照する
実在pathを意味しない。include先にある相対targetは、そのresource keyの親を基準に解決する。URLを
取得するか、cacheを使うか、認証を付与するかはhostの責務である。

== 対応directive

次を文書構造解析前に処理する。

* `include::target[]`
* `ifdef::name[]`、`ifndef::name[]`、`ifeval::[expression]`、`endif::[]`
* 行頭のbackslashによるdirectiveのliteral化

include属性は`tag`／`tags`、`lines`、`leveloffset`、`indent`およびUTF-8の`encoding`に対応する。
属性値の完全なAsciiDoc引用文法、optional include、wildcardおよびURI転送は未対応である。

== 安全性と上限

`PreprocessOptions`はinclude深さ、include件数、展開後総byte、展開nodeおよびsource map segment数の
上限を持つ。
循環、絶対path、path traversal、制御文字、未許可schemeを拒否する。`Secure` modeではscheme付き
targetを許可しない。他のmodeでも`allowed_schemes`に明示されたschemeだけをsnapshotから参照する。

コアはURLを取得しないため、timeout、redirect、response size、content type、認証情報および
network addressの制限はresourceをsnapshotへ格納する前にhostが強制する。コア側の総byte、件数、
深さ制限は取得済みresourceを用いた展開量を制限する第2の防壁である。

== source map

`PreprocessedDocument.source_map`は展開後の半開byte rangeを元の`SourceId`とrangeへ対応付ける。
`SourceMapping::Identity`は相対byte位置を保存し、indent、見出しlevel、escape解除またはinline
conditionalで変換された`WholeOrigin`は元の行全体へ保守的に対応する。rangeが複数segmentを横断する
場合、`origins_for_range`は順序付きの複数originを返す。

`PreprocessedAnalysis::project_origins`はdiagnostic、related information、fix、reference、resource、
document symbolを型付きoriginへ変換する。referenceとresourceは構文全体とtargetを別々に投影する。
複数originまたは変換segmentにかかるfixは`applicable = false`となり、hostは適用してはならない。
`ProjectionLimits`は1回のprojectionが生成できるorigin segment総数を制限する。include directiveには
記述位置の`source_id`とinclude先の`resource_source_id`を別々に保持するため、Document Linkと
Definitionを展開後offsetの推測なしに構築できる。

WASM adapterもmapping種別を含む同じsource mapを公開する。resource取得とsnapshot更新はLSP adapter
またはその上位hostが担い、コアへI/Oやasync runtimeを持ち込まない。
