= 現行契約の正本索引

この文書は、現行実装を説明する規範文書の入口である。作業の経緯はIssue、長期的な
設計判断はADR、将来の機能はロードマップを参照する。

初回公開に先立ち、開発中に増加していた6種類の契約versionを一度だけすべて`1`へresetした。これは公開前の
履歴整理であり、公開baseline確定後は番号を再利用または巻き戻さない。配布時の対応関係は
xref:release-distribution.adoc[GitHub Releases配布契約]を参照する。

== 文書の分類

このディレクトリは現行仕様を記述する。xref:adr/index.adoc[ADR]は採用済みの長期的な
設計判断、xref:roadmap.adoc[ロードマップ]は未実装の計画、xref:../issues/README.adoc[Issue]
は完了履歴と作業単位を記録する。Issue本文を現行仕様の正本として参照してはならない。
