== このリポジトリで使う言葉

通常の利用に必要な説明では、できるだけ日本語を使います。API名やプログラム内の名前は、
コードと一致させるため英字のまま表記します。

[cols="1,3",options="header"]
|===
|表記 |この文書での意味

|公開仕様
|ライブラリや配布物を利用するプログラムとの約束。以前の「contract」と同じ意味。

|対応範囲
|AdocWeaveが受け付け、期待どおりに処理するAsciiDocの範囲。以前の「profile」と同じ意味。

|意味情報
|解析して得た、見出し・リンク・表・属性などの構造化された情報。以前の「semantic model」や「projection」の説明で使っていた語。

|利用側アプリ
|AdocWeaveを呼び出すCLI、エディター、Webアプリなど。以前の「host」と同じ意味。

|テスト用入力
|期待する結果と組で保存する入力例。以前の「fixture」と同じ意味。

|===

厳密な用語が必要な箇所では、英語の名前をコード書式で併記します。
