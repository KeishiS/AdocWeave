= ADR 0002: ホスト拡張境界
:status: accepted

== 文脈

ノートアプリ固有の識別子、DB、権限およびURL生成を標準AsciiDoc解析へ組み込むと、
コアが特定製品へ依存する。

== 決定

コアは標準構文、汎用属性、通常リンクおよび型付きxrefを解析する。任意スキームの意味、
永続化、アクセス制御および解決済みURLはホストが解析後に付加する。

== 帰結

製品拡張は別リポジトリで実装できる。Parserへ任意コードを注入するplug-in ABIは提供せず、
拡張は公開Analysis、projectionおよびResolver契約だけを利用する。
