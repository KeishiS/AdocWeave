= AdocWeave browser package

このarchiveは、AdocWeaveのWebAssembly moduleとWeb Worker adapterを同じversionで提供する。
公的なpackage registryへは登録せず、リポジトリ所有者のGitHub Releasesから取得して展開する。

== 内容

* `wasm/adocweave_wasm.js`と`wasm/adocweave_wasm_bg.wasm`：`wasm-bindgen`のweb向け成果物
* `wasm/adocweave_wasm.d.ts`：生成された場合のTypeScript declaration
* `worker/index.mjs`と`worker/index.d.mts`：frontend向けの公開ES module入口と型定義
* `worker/client.mjs`、`worker/controller.mjs`、`worker/worker.mjs`：最新generationだけを採用する内部実装
* `worker/contracts.mjs`：公開パッケージ版
* `worker/protocol.generated.mjs`と`worker/protocol.generated.d.mts`：生成された公開型と版数

== 最小例

[source,javascript]
----
import { AdocWeaveClient, defaultAssetUrls } from "./worker/index.mjs";

const client = new AdocWeaveClient({
  ...defaultAssetUrls(),
  onResult(result) {
    // HTMLの信頼方針はホストが決める。この例では文字列として表示する。
    preview.textContent = result.html;
  },
  onError(error) {
    console.error(error.code, error.message);
  },
});
client.update({ version: 1, source: "= Title\n\nAsciiDoc text" });
----

公開入口がWorkerの生成、初期化、debounce、generation、cancelおよびdisposeを所有する。
`SharedArrayBuffer`を利用できない場合はWorker再生成へ自動的に縮退する。DOMへの反映とUI stateは
利用側が所有する。動作するtextarea例は`example/index.html`に含む。APIと返却値の詳細は
リポジトリの`docs/wasm-worker.adoc`を参照する。
